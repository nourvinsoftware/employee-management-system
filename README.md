# Employee Management System - Java

A Java-based employee management system created to practice object-oriented programming and basic business logic.

## Features

- Add employee information
- Manage employee ID
- Store department information
- Calculate salary
- Display employee details
- Organize code using multiple Java classes

## Technologies Used

- Java
- Git
- GitHub
- VS Code
